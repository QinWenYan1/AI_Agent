# CLAUDE.md

## 仓库定位

这是个人的 **AI Agent 学习笔记仓库**。使用结构化 Markdown 笔记记录 AI 开发工具和技术的知识，便于复习和检索。

## 目录结构

```
README.md                   # 根导航中枢——列出所有学习主题
<topic-name>/               # 每个主题一个文件夹
  README.md                 # 主题内导航——列出该主题的所有笔记
  0X-description.md         # 结构化笔记（每个主题不超过 3 篇）
```

## 笔记格式规范

- **中文为主**，关键术语用反引号括注英文：\`中文(English)\`
- 每篇笔记遵循固定模板：核心概念总览（含锚点跳转）→ 知识点（理论 + 示例 + 注意点）→ 核心要点总结 → 实践速查
- 每个知识点使用 `<a id="idN"></a>` 锚点，总览中通过 `[知识点N: 名称](#idN)` 链接
- 注意点使用 emoji 标签：⚠️ 警告 | 💡 技巧 | 🔄 关联 | 📋 术语
- 笔记之间的引用使用相对路径：`[[../02-workflows.md#id7]]`

## 添加新主题

1. 新主题只需一个文件夹 + 一个 `README.md` 导航 + 不超过 3 篇 `.md` 笔记
2. 在根 `README.md` 的"笔记目录"中添加新主题的入口
3. 遵循相同的笔记格式规范
